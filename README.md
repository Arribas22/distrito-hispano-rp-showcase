# Distrito Hispano RP - Technical Showcase

Curated source excerpt of a larger private framework for Arma Reforger roleplay servers.

## Scope of this repository
This is NOT the full production framework.
It is a selected technical showcase containing representative systems and coding patterns for review by recruiters and technical leads.

Included areas:
- Core runtime utilities and server init/security components
- Admin management structures (whitelists, ban and pending workflows)
- Inventory and vehicle-related components
- Account and persistence-oriented feature excerpts

## Why partial publication
The complete framework contains proprietary gameplay/business logic currently used in production. This public repo is designed to prove engineering capability without exposing full intellectual property.

## Copilot-driven development
The showcased systems were developed with GitHub Copilot as an implementation copilot, with manual architecture control, debugging and production hardening by the author.
